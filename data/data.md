### The data directory should be organized like this
```
├── data
│   ├── KITTI05
│   │   ├── imgs          # bev images 
│   │   ├── velodyne      # point clouds
|   |   └── pose.txt      # ground truth pose
|   |
|   |
│   └── Your dataset      # remember to custom your dataloader in dataset.py       
│       ├── imgs          # bev images generated by the script gen_bev_images.py
|       ├── velodyne      # your point clouds
|       └── pose.txt      # ground truth pose
```
